# RSPortfolioSite

Richard Sepši - personal portfolio website. Made to show my work in one place.

I's a simple frontend only website made with plain JS, CSS and HTML. It can be ran locally by just opening one of the html files in a browser, but search will not work unless it's ran on a live server.
The website doesn't collect any data and only saves language preferences into cookies.

Please report all bugs, typos and other problems by opening an issue here on github: https://github.com/RichardSepsi/RSPortfolioSite/issues/new
